Python example:

Алгоритм, представленный в задании:
def isEven(value):return value%2==0
+ Простой для восприятия при прочтении кода алгоритма,
- медленнее побитовых операций.


Алгоритм, реализованный в качестве ответа:
def is_even(value):
    if value & 1 == 0:
        return True
    else:
        return False

- Для восприятия несколько сложнее, чем тот, что представлен в задании,
+ быстрее алгоритма, приведенного в задании.
